# Portfolio Project

## Setup
This is a static, mobile-first portfolio built with HTML, CSS, and JavaScript.
No build tools or frameworks are required.

## Technologies
- HTML5 (semantic markup)
- CSS3 (custom properties, Grid, Flexbox)
- JavaScript (navigation and form validation)

## Design Decisions
- Mobile-first responsive layout starting at 320px
- Reusable components for buttons, cards, and layout
- Accessibility-first approach following WCAG 2.1 AA guidelines
- Lightweight performance-focused implementation

## Deployment
The site is deployed using GitHub Pages and can be updated
by pushing changes to the repository.
